\# 联邦学习诈骗检测实验报告



\## 项目简介



本项目基于Flower框架实现联邦学习诈骗检测模型。



\## 实验环境



\- Python 3.10

\- PyTorch

\- Transformers

\- Flower



\## 模型



\- DistilBERT



\## 联邦配置



\- Client数量：2

\- 聚合算法：FedAvg

\- 通信轮数：3



\## 实验结果



Accuracy  : 0.9375



Precision : 0.8889



Recall : 1.0000



F1-score : 0.9412



\## 结论



模型能够有效识别诈骗文本，在测试集上取得较好的检测效果。
